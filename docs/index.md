# USDAgent

> Orchestrate USD scenes via natural language.

```bash
pip install usdagent
usdagent run --stage scene.usda "Add 50 shelf units to a warehouse floor"
```

## Features

- Natural language → USD operations
- Procedural placement with variation (scatter, randomize)
- Material/shader assignment
- Scene validation
- Works headless — no Omniverse required
- Optional Omniverse Kit live-preview extension
- CLI + Python API
